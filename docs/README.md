# Math formulas
## Area
- Circle: S = πR²
- Rectangle: S = a*b
- Square: S = a²
- Triangle: S = a*h/2

## Perimeter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a
- Triangle: P = a+b+c
